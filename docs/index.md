# Docs

Student-facing tutorials for the HAP 404 project. Concepts build on each other — reading order is recommended.

1. [CORS is hard and everyone hates it](cors-is-hard-and-everyone-hates-it.md) — Why CORS exists, how the browser enforces it, the five errors you'll actually see, and a hands-on exercise using this repo.
2. [Local debugging with DevTools and netlify dev](local-debugging-with-devtools-and-netlify-dev.md) — Setup, the two-window dev workflow, reading netlify dev output, DevTools tabs, and walkthroughs of six real failure modes.
3. [Hardening walkthrough — applying each audit fix](hardening-walkthrough-applying-each-audit-fix.md) — Read-along explaining what was changed and why: CSP, fail-closed CORS, rate limiting, origin checks, and defensive response parsing.
4. [Secretlint — keeping secrets out of git](secretlint-keeping-secrets-out-of-git.md) — Why leaked API keys are dangerous, how secretlint blocks accidental commits, and what to do when it fires.
